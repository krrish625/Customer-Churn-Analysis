# Customer Churn Analysis & Prediction

## Project Overview
This project predicts customer churn using Machine Learning techniques.

## Steps Performed
- Data Preparation
- Feature Selection
- Train-Test Split
- Model Training using Logistic Regression
- Model Evaluation

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Results
- Accuracy: ~78%
- ROC-AUC Score: 0.69
